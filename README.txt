
This python program lists the airports within a user defined radius of a user defined position.
To be able to run, install Cloudant python library from here first:
https://github.com/cloudant/python-cloudant.

Usage:
  CloudantProj.py <pos.lat> <pos.long> <distance>

Where
  <pos.lat>: The latitude of the center in degrees. (-90.0 - 90.0)
  <pos.long>: The longitude of the center in degrees. (0.0 - 360.0)
  <distance>: The radius in kilometers.
